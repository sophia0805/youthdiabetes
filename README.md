# youthdiabetes.ai — Mobile Style Demo ✅

This is a small, mobile-first website template that reproduces a modern health-site style while using original placeholder content.

## Files
- `index.html` — Home (style-driven demo)
- `risk.html` — Risk sample layout
- `resources.html` — Resources listing layout
- `about.html` — About / design notes
- `css/style.css` — Mobile-first stylesheet
- `js/main.js` — Minimal JS for nav
- `assets/logo.svg` — Placeholder logo

## Preview locally
Option A — open files directly:
- Open `index.html` in your browser (works for visual review)

Option B — run a local static server (recommended):
- Using Node (dev server): `npm install` then `npm run dev`
- To build a production bundle: `npm run build` then `npm run preview` to locally preview the built site
- Alternative simple server: `python -m http.server 8000` (serves built files from a `dist` folder)

Then visit `http://localhost:3000` (dev) or `http://localhost:8000` (preview) and click pages.

## Notes
- This template intentionally keeps style (colors, spacing, components) and uses original placeholder content only.
- Feel free to customize colors in `css/style.css` and adjust layout breakpoints.
